# sg-vditor

致力于打造精简而不失拓展的 SVG 的编辑器。

# SgVditor 方法

| 方法名      |                   作用                   |
|----------|:--------------------------------------:|
| listNodes |                 列出所有图形                 |
|cleanNodes|                  清空画布                  |
| addRect  |                  添加矩形                  |
| addPolygon |                 添加多边形                  |
| updateType         | 变更画笔类型（select / rect / line / polygon） |
